# PORTFOLIO
This is my personal website created for the purpose of showcasing or displaying completed, upcoming or in progress projects. It also a nice place to learn more about me and what I can do. The form section and the socials are another ways of bring users or visitors closer. This project was achieved using the following technologies :

- HTML5
- CSS3
- Bootstrap
- JavaScript and
- GSAP
